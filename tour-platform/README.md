# Веб-платформа для бронирования и управления турами

## Описание проекта
Веб-приложение для поиска, бронирования и управления турами. 
Платформа предоставляет удобный интерфейс для клиентов и мощные инструменты для администраторов.

## Технологии
- **Backend**: PHP 8.1 (без фреймворков)
- **Frontend**: HTML5, CSS3, JavaScript
- **База данных**: MySQL 8.0
- **Веб-сервер**: Nginx/Apache
- **Контейнеризация**: Docker, Docker Compose

## Функциональность

### Для гостей:
- Просмотр списка туров
- Поиск и фильтрация туров по странам и датам
- Регистрация в системе

### Для клиентов:
- Вход в личный кабинет
- Бронирование туров
- Просмотр истории заказов
- Отмена бронирования
- Управление личными данными

### Для администраторов:
- Управление турами (CRUD)
- Управление заказами
- Просмотр статистики
- Генерация отчетов
- Мониторинг системы
- Резервное копирование БД
