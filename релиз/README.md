# 🌍 Веб-платформа для бронирования и управления турами

---

## 📋 Описание проекта

Веб-приложение для поиска, бронирования и управления туристическими поездками.

**Проблемы, которые решает:**
- Информационная фрагментация — поиск туров на множестве сайтов
- Сложность сравнения цен и условий
- Отсутствие единого личного кабинета для управления бронированиями

**Роли пользователей:**
- **Гость** — просмотр туров, регистрация
- **Клиент** — бронирование, личный кабинет, история заказов
- **Администратор** — управление турами, заказами, отчётами

## 🚀 Установка и запуск

### Способ 1: Docker (рекомендуемый)

```bash
# Клонирование репозитория
git clone https://github.com/2Dbaca/Web-platform-for-booking-and-managing-tours-.git
cd Web-platform-for-booking-and-managing-tours-

# Запуск контейнеров
docker-compose up -d
```

### Способ 2: XAMPP

1. Скопировать папку проекта в `C:\xampp\htdocs\tour-platform`

2. Запустить Apache и MySQL в XAMPP Control Panel

3. Импортировать `database.sql` через phpMyAdmin

4. Открыть `http://localhost/tour-platform`

# Остановка

```bash
docker-compose down
```
